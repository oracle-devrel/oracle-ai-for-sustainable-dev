# Oracle AI Database with AWS Bedrock

This directory contains examples and implementations for integrating Oracle Database with AWS Bedrock for AI and machine learning applications.

## Overview

This project demonstrates how to leverage Oracle Database AI capabilities with AWS Bedrock's foundation models to build intelligent applications.

## Features

- Integration with AWS Bedrock foundation models
- Oracle Database AI Vector Search
- RAG (Retrieval Augmented Generation) implementations
- AI-powered database operations

## Prerequisites

- Oracle Database 26ai or later
- AWS Account with Bedrock access
- Python 3.8+
